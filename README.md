# Curriculum Vitae y Portafolio — Adolfo Antequera

Este proyecto es una prueba para el cierre de la Unidad 1 del **Bootcamp Full Stack JavaScript** de Desafío Latam. El objetivo es presentar un portafolio web y un currículum vitae digital, aplicando conocimientos de HTML y CSS, y sentando las bases para futuros desarrollos en JavaScript y frameworks modernos.

## Descripción General

El repositorio contiene una web personal compuesta por:

- **Currículum Vitae (index.html):** Presenta la información profesional, habilidades, experiencia y educación de Adolfo Antequera en un formato visual atractivo y moderno.
- **Portafolio (portafolio.html):** Muestra proyectos destacados, habilidades técnicas y enlaces a trabajos previos, con un diseño responsivo y visualmente llamativo.
- **Contacto (contacto.html):** Formulario simple para contacto, estilizado y funcional a nivel de frontend.

## Estructura del Proyecto

```
├── index.html           # Página principal: CV
├── portafolio.html      # Página de portafolio de proyectos
├── contacto.html        # Página de contacto
├── colores              # Paleta de colores utilizada
├── assets/
│   ├── css/
│   │   ├── style.css         # Estilos del CV
│   │   ├── portafolio.css    # Estilos del portafolio
│   │   └── contacto.css      # Estilos del formulario de contacto
│   ├── img/                  # Imágenes usadas en el CV y portafolio
│   └── media/                # Archivos multimedia (videos, etc.)
└── README.md            # Este archivo
```

## ¿Cómo ver el proyecto?

1. **Clona o haz fork de este repositorio:**
	 - Puedes hacer un fork para trabajar sobre tu propia copia.
	 - O clonar con:
		 ```
		 git clone <url-del-repositorio>
		 ```
2. **Abre el archivo `index.html` en tu navegador** para ver el CV.
3. Navega a `portafolio.html` para ver los proyectos y a `contacto.html` para el formulario de contacto.

## Tecnologías Utilizadas

- Google Fonts y FontAwesome para iconografía y tipografía
